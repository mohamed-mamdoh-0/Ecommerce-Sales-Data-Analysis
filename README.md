# 📊 E-commerce Sales & Logistics Data Analysis Dashboard

Welcome to the **E-commerce Sales & Logistics Analysis Project**. This project provides a comprehensive data analytics solution built entirely inside **MS Excel**, focusing on connecting financial KPIs with supply chain efficiency and customer satisfaction.

---

## 🛠️ Key Features & Tools Used
* **Advanced Data Modeling:** Built a complex data model to handle large-scale datasets seamlessly using **Power Query (ETL)** and **Power Pivot**.
* **Dynamic Analytics:** Implemented dynamic KPI switching for Revenue, Net Profit, and Shipping Costs using advanced **DAX Formulas**.
* **Professional UI/UX:** Designed an interactive dashboard with a sleek dark theme and functional Navigation Bars.
